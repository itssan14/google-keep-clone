## Google Keep

---

Live version hosted on Vercel
[here](https://google-keep-clone-delta.vercel.app/)

---

### Setup Instructions

- Install dependencies with - `npm install --save` or `yarn add`
- To start running the development version - `npm run dev`
- The local version starts [here](http://localhost:8080)

---

### Requirements

- [x] Filters / Data persistence on page refresh
- [x] Global search
- [x] Create sections for different types of notes - pinned, notes and archived
- [x] Persist edit modal on page refresh
- [x] Usage of external UI libraries is discouraged

### Bonus Feature

- [x] Mobile Responsive
- [x] Supports dark / light mode
- [ ] Markdown and image support

### Tools Used

- Next.js
- React.js
- Redux (with Redux Toolkit)
- Typescript
